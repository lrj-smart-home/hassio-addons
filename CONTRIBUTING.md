# Contributing

We are open to, and grateful for, any contributions made by the community. By
contributing to this add-on, you agree to abide by the
[code of conduct](https://github.com/lrj-smart-home/hassio-addons/blob/main/CODE_OF_CONDUCT.md).

## Reporting Issues and Asking Questions

Before opening an issue, please search the
[issue tracker](https://github.com/lrj-smart-home/hassio-addons/issues) to make
sure your issue hasn't already been reported.

## Submitting Changes

- Open a new issue in the
  [Issue tracker](https://github.com/lrj-smart-home/hassio-addons/issues).
- Fork the repo.
- Create a new branch based off the `main` branch.
- Submit a pull request, referencing any issues it addresses.

Please try to keep your pull request focused in scope and avoid including
unrelated commits.

After you have submitted your pull request, we'll try to get back to you as soon
as possible. We may suggest some changes or improvements.

Thank you for contributing!
